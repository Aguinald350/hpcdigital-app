# HPCDIGITAL

HPCDIGITAL is a Flutter-based mobile application designed to centralize church and community information into one accessible digital platform.

## Overview

The app provides users with structured access to multilingual hymn content, church information, leadership data, events, and community-related resources.

It was built to make important institutional and spiritual content easier to access, manage, and distribute through a mobile experience.

---

## Problem It Solves

In many communities, important information is often scattered across paper documents, social media groups, and informal communication channels.

HPCDIGITAL helps solve this by creating a centralized and organized mobile platform where users can easily access:

- Hymns by language and section
- Event schedules
- Church and leadership information
- Community structure and navigation

---

## Key Features

### User Features
- Browse hymns by section and language
- Search hymns by title or number
- View hymn details
- Access church and leadership information
- View event schedules and categories
- Navigate structured church/community data

### Admin Features
- Admin authentication
- Add and manage hymn content
- Create and edit events
- Register church structures (districts, intendências, churches)
- Manage community-related content
