Política de Privacidade — Hinário Povo Cantai / HPC Digital

Última atualização: 26 de outubro de 2025

Ao usar o aplicativo HPC, você concorda com a coleta e uso de informações conforme descrito nesta política.

Introdução

Antes de usufruir dos nossos serviços, é importante que você saiba quais dados coletamos e como os utilizamos.

Ao optar por usar nossos serviços HPC, você concorda com a coleta e uso das informações. Os dados coletados são usados primordialmente para fornecer e melhorar os serviços.

Dados Tratados e Finalidade do Tratamento

Podemos solicitar dados como:

  • Nome completo e e-mail
  • Telefone
  • Data de nascimento, sexo, cidade, localização
  • Comentários, opiniões e interações no aplicativo
  • Dados de pagamento quando necessário

Finalidades:

  • Identificação e liberação de acesso
  • Login com telefone (SMS)
  • Comunicações importantes
  • Processamento de pagamentos quando aplicável

Você pode excluir seu perfil a qualquer momento pelo app.

Coletamos também dados técnicos como IP, tipo de dispositivo, SO, versão do app e data/hora de uso. Esses dados podem ser processados fora de Angola em conformidade com esta política.

Compartilhamento de Dados

Podemos compartilhar dados com terceiros para facilitar serviços, executar funções ou analisar uso. Esses terceiros só podem usar os dados para finalidades definidas contratualmente.

Podemos compartilhar informações para cumprir leis, ordens judiciais, prevenir fraude ou proteger direitos e segurança de usuários.

Links para Outros Sites

Ao acessar sites de terceiros a partir do app, você estará sujeito às políticas desses sites.

Cookies

O app não usa cookies diretamente, mas serviços de terceiros podem utilizá-los. Você pode recusar cookies, ciente de que isso pode afetar algumas funcionalidades.

Segurança

Usamos medidas razoáveis de segurança, incluindo criptografia em trânsito. Nenhum método é 100% seguro, mas notificaremos incidentes relevantes se ocorrerem.

Direitos dos Titulares

Você pode acessar, corrigir e excluir sua conta pelo aplicativo. Podemos solicitar verificação de identidade.

Privacidade Infantil

O serviço não é destinado a menores de 12 anos. Caso dados de crianças sejam informados, serão removidos ao identificarmos o fato.
